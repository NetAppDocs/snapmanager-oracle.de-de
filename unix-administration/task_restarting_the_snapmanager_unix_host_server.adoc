= Starten Sie den SnapManager UNIX-Hostserver neu
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den Server auf einem UNIX-Host mithilfe der CLI neu starten.

. Geben Sie zum Neustart des Servers den folgenden Befehl ein:
`smo_server restart`
