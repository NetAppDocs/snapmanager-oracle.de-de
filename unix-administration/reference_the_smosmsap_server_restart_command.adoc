= Der Befehl smo_Server restart
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit diesem Befehl wird der SnapManager-Hostserver neu gestartet und als root eingegeben.



== Syntax

[listing]
----

        smo_server restart
[-quiet | -verbose]
----


== Parameter

* *-ruhig*
+
Gibt an, dass nur Fehlermeldungen auf der Konsole angezeigt werden. StandardmÃ¤ÃŸig werden Fehler- und Warnmeldungen angezeigt.

* *-verbose*
+
Gibt an, dass Fehler-, Warn- und Informationsmeldungen auf der Konsole angezeigt werden.





== Beispielbefehl

Im folgenden Beispiel wird der Host-Server neu gestartet.

[listing]
----
smo_server restart
----
